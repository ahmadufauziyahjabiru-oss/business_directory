# Local Business Directory 🏢

## What the app does
[span_1](start_span)This is a web application built with Flask that serves as a directory for local startups and businesses[span_1](end_span). 
* **[span_2](start_span)OOP**: It uses a `Business` class to manage data[span_2](end_span).
* **[span_3](start_span)Data Structures**: It uses a **Stack** to display the "Recently Added" businesses so that the newest entries appear first[span_3](end_span).
* **[span_4](start_span)[span_5](start_span)Timestamps**: Every entry is automatically timestamped using Python's `datetime` module[span_4](end_span)[span_5](end_span).

## How to run the Flask app locally
Follow these simple steps to get the app running on your computer:

1. **Install Flask**: Open your terminal and run:
   [span_6](start_span)`pip install Flask`[span_6](end_span)
2. **Download the code**: Ensure `app.py`, `models.py`, and the `templates` folder are in the same directory.
3. **Run the server**: In the terminal, type:
   [span_7](start_span)`python app.py`[span_7](end_span)
4. **[span_8](start_span)View the app**: Open your web browser and go to `http://127.0.0.1:5000`[span_8](end_span).
